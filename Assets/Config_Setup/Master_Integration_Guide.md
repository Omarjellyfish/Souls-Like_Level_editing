# Master Setup & Integration Documentation Folder

This folder has been broken down into modular sections to match the modular design of the codebase.

*   **`02_Combat_Stats_Guide.md`** -> Read this to understand how `Health`, `Stamina`, and `Poise` interlock to create Parries, Dodges, Staggers, and Hyper Armor.
*   **`03_Camera_Targeting.md`** -> Read this for the specifics on the `LockOnManager` and the dual-Cinemachine toggling system.
*   **`Animation_Setup/Animator_Combos_Guide.md`** -> Read this for the step-by-step Unity Editor setup for configuring Sub-State Machines (and why Blend Trees will break combat!).
